# koda-agent (DEPRECATED)

> **This crate has been renamed.** Use [`koda-cli`](https://crates.io/crates/koda-cli) instead.

```bash
cargo install koda-cli
```

The Koda project has been split into two crates:
- **`koda-core`** — engine library
- **`koda-cli`** — CLI binary (installs the `koda` command)

See [github.com/lijunzh/koda](https://github.com/lijunzh/koda) for details.
